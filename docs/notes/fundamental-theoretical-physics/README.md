## 前言
本篇笔记是北京大学《理论物理基础》课程的笔记。这门课是24届刚刚开始开设的课程，可以被视为正式四大力学的先修课，但事实上其内容已经基本涵盖了四大力学的大部分内容。

该课程分为上下两部分，用两学期的时间完成，上半学期学习理论力学与电动力学，下半学期学习热统和量子力学。上半学期的授课老师是孟策老师，授课风格比较适合竞赛生，会两个小时不停地板书推导，
如果电波对上的话听起来非常的爽，相对而言考试难度也会比较高，班上基本都是卓越生。
