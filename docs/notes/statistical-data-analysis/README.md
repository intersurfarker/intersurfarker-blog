## 概述

本目录包含统计数据分析相关的学习笔记，涵盖概率论基础、常用概率分布以及概率论的逻辑基础等内容。这些笔记旨在为物理实验数据分析、统计推断等应用提供理论基础。

## 笔记列表

### 基础概念
- **[基本概念](r9r806xh/)**：概率论的基础知识，包括概率公理、随机变量、条件概率、贝叶斯定理、概率密度函数、期望值、方差、协方差、误差传递和随机函数的正交变换等核心概念。

### 概率分布
- **[常用概率函数](0n9zym1x/)**：详细介绍常用的概率分布，包括二项分布、多项分布、泊松分布、均匀分布、指数分布、高斯分布、对数正态分布、卡方分布和柯西分布等，涵盖其定义、性质和应用场景。

### 理论深入
- **[概率论沉思录](qkeu90ko/)**：探讨概率论的逻辑基础，包括布尔代数、完备集、合情推理（Plausible Reasoning）、乘法规则和加法规则等，从哲学和逻辑角度理解概率论的基本原理。

## 学习路径建议

1. 从**基本概念**开始，建立概率论的基础知识框架。
2. 学习**常用概率函数**，掌握各种概率分布的特性和应用。
3. 阅读**概率论沉思录**，深入理解概率论的逻辑基础和哲学思考。

## 相关应用

这些笔记内容为以下领域提供了理论基础：
- 物理实验数据分析与误差处理
- 统计推断与假设检验
- 机器学习中的概率模型
- 随机过程与蒙特卡洛方法
