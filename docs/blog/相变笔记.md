# 相变笔记
## 化学势
考虑与环境可以有粒子交换的系统，如果向系统中加入一个粒子而不改变系统的体积或熵，则系统的内能会改变一个量，我们定义这个改变量为**化学势**。因此，热力学基本方程将会有以下修正：
$$\mathrm{d}U=T\mathrm{d}S-p\mathrm{d}V+\mu\mathrm{d}N$$
这给出了以下微分关系：
$$\mu=\left(\frac{\partial U}{\partial N}\right)_{S,V}$$
我们也可以通过勒让德变换容易获得以下关系：
$$\mathrm{d}H = T\mathrm{d}S + V\mathrm{d}P + \mu \mathrm{d}n \implies \mu = \left( \frac{\partial H}{\partial n} \right)_{S, P}$$
$$\mathrm{d}F = -S\mathrm{d}T - P\mathrm{d}V + \mu \mathrm{d}n \implies \mu = \left( \frac{\partial F}{\partial n} \right)_{T, V}$$
$$\mathrm{d}G = -S\mathrm{d}T + V\mathrm{d}P + \mu \mathrm{d}n \implies \mu = \left( \frac{\partial G}{\partial n} \right)_{T, P}$$
就化学系统而言，最后一个方程是最有用的，因为通常$T,P$不变的条件很方便满足。
## 热动平衡条件
将一个系统的熵视作是$U,V,N$的一个函数，因此可以写出：
$$\mathrm{d} S=\left( \frac{\partial S}{\partial U}\right )_{N,V}\mathrm{d} U+\left( \frac{\partial S}{\partial V}\right )_{N,U}\mathrm{d} V+\left( \frac{\partial S}{\partial N}\right )_{U,V}\mathrm{d} N$$
将热力学基本方程改写为：
$$\mathrm{d} S=\frac{\mathrm{d} U}{T}+\frac{p\mathrm{d} V}{T}-\frac{\mu \mathrm{d} N}{T}$$
比较两式，我们得到：
$$\left(\frac{\partial S}{\partial U}\right )_{N, V}=\frac{1}{T}, \left( \frac{\partial S}{\partial V}\right )_{N,U}=\frac{p}{T},\left( \frac{\partial S}{\partial N}\right )_{U,V}=-\frac{\mu}{T}$$
考虑两个与外界热孤立且总体积恒定，但可以彼此交换热量，改变体积，交换粒子的系统。此时平衡条件要求体系的熵$S$极大，或者：
$$\mathrm{d}S=0$$

如果两个系统之间交换$\mathrm{d} U,\mathrm{d} V,\mathrm{d} N$，则总熵的变化值为：
$$\begin{aligned}
    \mathrm{d} S=&\mathrm{d} S_1+\mathrm{d} S_2 \\
    =&\left(\frac{1}{T_1}\mathrm{d} U_1+\frac{p_1}{T_1}\mathrm{d} V_1-\frac{\mu_1}{T_1}\mathrm{d} N_1\right)+\left(\frac{1}{T_2}\mathrm{d} U_2+\frac{p_2}{T_2}\mathrm{d} V_2-\frac{\mu_2}{T_2}\mathrm{d} N_2\right) \\
    =&\left(\frac{1}{T_1}-\frac{1}{T_2}\right)\mathrm{d} U+\left(\frac{p_1}{T_1}-\frac{p_2}{T_2}\right)\mathrm{d} V-(\frac{\mu_1}{T_1}-\frac{\mu_2}{T_2})\mathrm{d} N
\end{aligned}$$
可见，平衡条件为：
$$\mathrm{d} S=0\Rightarrow T_1=T_2,p_1=p_2,\mu_1=\mu_2$$
具体地，第一个条件对应了热平衡条件。如果不满足，则有：
$$\mathrm{d} S=\left(\frac{1}{T_1}-\frac{1}{T_2}\right)\mathrm{d} U_1>0$$
倘若$T_1>T_2$，则有$\mathrm{d} U_1<0$，也即热量从1传给2。

第二个条件对应了力学平衡条件，如果不满足，假设两系统已经达成了热平衡条件，则有：
$$\mathrm{d} S=\frac{1}{T}T(p_1-p_2)\mathrm{d} V_1>0$$
倘若$p_1>p_2$，则有$\mathrm{d} V_1>0$，也即压强高的系统膨胀。

第三个条件对应了**相变平衡条件**，如果不满足，假设两系统已经达成了热平衡条件，则有：
$$\mathrm{d} S=-\frac{1}{T}(\mu_1-\mu_2)\mathrm{d} N_1>0$$
倘若$\mu_1>\mu_2$，则有$\mathrm{d} N_1<0$。也即：粒子从化学势高的一方转移到化学势低的一方。这清楚地阐述了化学势的物理意义。

值得一提的是，倘若体系的粒子数不守恒，那么$\mathrm{d} N_1和\mathrm{d} N_2$将不再独立，而$\mathrm{d} S=0$会要求$\mu_1=\mu_2=0$。也即：**粒子数不守恒系统的化学势等于0**.

## 化学势与吉布斯函数
化学势有一个特别的性质，如果一个系统的大小改变一个因子$\lambda$，那么我们预期这个系统的所有广延量会按照$\lambda$改变大小。将$S$写作$U,V,N$的函数，它们都是广延量，于是：
$$\lambda S(U,V,N)=S(\lambda U,\lambda V,\lambda N)$$
由齐次函数的欧拉定理，可以得到：
$$\begin{aligned}
    S=&\frac{\partial S}{\partial U}U+\frac{\partial S}{\partial V}V+\frac{\partial S}{\partial N}N \\
    =& \frac{U+pV-\mu N}{T}
\end{aligned}$$
化简得到：
$$\mu N=U-TS+pV=G$$
因此，我们得到了：
$$\boxed{\mu=\frac{G}{N}}$$
也即：**化学势等于单粒子吉布斯函数**。

还有一种常用的形式是作勒让德变换，使得自然变量从$(T,V,N)$变为$(T,V,\mu)$，得：
$$\mathrm{d}(F-\mu N)=-S\mathrm{d} T-p\mathrm{d} V-N\mathrm{d} \mu$$
我们引入**巨势**$\Psi$：
$$\Psi=F-\mu N=U-TS-\mu N=F-G=-pV$$
$$\mathrm{d}\Psi=-S\mathrm{d} T-p\mathrm{d} V-N\mathrm{d} \mu$$
巨势没有明确的热力学意义，但它直接联系着巨配分函数。

我们可以将以上定律推广到多种类型粒子的系统，即有：
$$\mathrm{d} U=T\mathrm{d} S-p \mathrm{d} V+\sum_i\mu_i\mathrm{d} N_i$$
$$\mathrm{d} F=-p\mathrm{d} V-S\mathrm{d} T+\sum_i\mu_i\mathrm{d} N_i$$
$$\mathrm{d} G=V\mathrm{d} p-S\mathrm{d} T+\sum_i\mu_i\mathrm{d} N_i$$
特别地，当压强和温度保持为常数时，即有：
$$\mathrm{d} G=\sum_i \mu_i\mathrm{d} N_i$$

## 相变潜热
对于在临界温度$T_c$达到热力学平衡的两个相，通常情况下，可以发现在恒定温度下$T_c$下从相1变化到相2，需要提供额外的热量，这被称为潜热，它由下式给出：
$$L=\Delta Q_{rev}=T_c(S_2-S_1)$$
值得注意的是，相变潜热实际上对应的是两相边界点上的焓变，这是因为由：
$$\mathrm{d} H=T\mathrm{d} S+V\mathrm{d} p$$
在恒压恒温条件下，即有：
$$\Delta H=T_c\Delta S=L$$

## 相平衡条件
考虑任意两个相，当系统的温度和压强保持不变时，吉布斯函数是应该取极小值的量，即有：
$$\mathrm{d} G=\mu_1\mathrm{d} N_1+\mu_2 \mathrm{d} N_2=0$$
又考虑到$\mathrm{d} N_1=-\mathrm{d} N_2$，便有：
$$\mu_1=\mu_2$$
推广到多相的情形，我们知道了**在相平衡时，每个共存相都有相同的化学势**。

## 克劳修斯-克拉珀龙方程
现在我们想要在$p-T$平面上找到描述相界的方程，这条两相共存线由以下方程决定：
$$\mu_1(p,T)=\mu_2(p,T)$$
沿着这条相界移动，必须有：
$$\mathrm{d} \mu_1=\mathrm{d} \mu_2$$
由于化学势是单粒子吉布斯自由能，我们可以知道：
$$\mathrm{d}\mu=-s\mathrm{d} T+v\mathrm{d} p$$
其中$s,v$分别代表单粒子的熵和体积。因此，我们有：
$$-s_1\mathrm{d} T+v_1\mathrm{d} p=-s_2\mathrm{d} T+v_2\mathrm{d} p$$
这里已经用到了相平衡时温度和压强平衡的条件，整理得到：
$$\frac{\mathrm{d} p}{\mathrm{d} T}=\frac{s_1-s_2}{v_2-v_1}$$
定义单粒子的相变潜热$l=T\Delta s$，则有：
$$\frac{\mathrm{d} p}{\mathrm{d} T}=\frac{l}{T(v_2-v_1)}$$
或者也可以等价地写为：
$$\boxed{\frac{\mathrm{d} p}{\mathrm{d} T}=\frac{L}{T(V_2-V_1)}}$$
这被称为**克劳修斯-克拉珀龙方程**。

## 蒸汽压方程
在得到共存线的斜率关系后，我们自然关心$p,T$具体的函数关系。值得注意的是，公式中的$L$一般依赖于$T$。一种推导方法如下，考虑沿共存线$L/T$随$T$的变化率：
$$
\begin{aligned}
    \frac{\mathrm{d}}{\mathrm{d} T}\left(\frac{L}{T}\right)=&\frac{\partial}{\partial T}\left(\frac{L}{T}\right)_p+\frac{\partial}{\partial p}\left(\frac{L}{T}\right)_T\times\frac{\mathrm{d} p}{\mathrm{d} T} \\
    =&\left(\frac{\partial \Delta S}{\partial T}\right)_p+\left(\frac{\partial \Delta S}{\partial p}\right)_T\times\frac{\mathrm{d} p}{\mathrm{d} T} \\
    =&\frac{C_{p,v}-C_{p,l}}{T}-\left[\frac{\partial(V_v-V_l)}{\partial T}\right]_p\times\frac{\mathrm{d} p}{\mathrm{d} T}
\end{aligned}
$$
在气液相变下，有$V_v\gg V_l$，并代入理想气体方程$pV_v=RT$和克拉珀龙方程$\frac{\mathrm{d} p}{\mathrm{d} T}=\frac{L}{T(V_v-V_l)}$，我们得到：
$$\frac{\mathrm{d} }{\mathrm{d} T}\left(\frac{L}{T}\right)=\frac{C_{p,v}-C_{p,l}}{T}-\frac{L}{T^2}$$
而等号左边拆开为：
$$\frac{\mathrm{d} }{\mathrm{d} T}\left(\frac{L}{T}\right)=\frac{1}{T}\frac{\mathrm{d} L}{\mathrm{d} T}-\frac{L}{T^2}$$
对比两式，得到：
$$\frac{\mathrm{d} L}{\mathrm{d} T}=C_{p,v}-C_{p,l}$$
如果假设热容变化不大，则有：
$$L=L_0+(C_{p,v}-C_{p,l})T$$
代入克拉珀龙方程：
$$\frac{\mathrm{d} p}{\mathrm{d} T}=\frac{L}{TV_v}=\frac{pL}{RT^2}=\frac{p}{RT^2}[L_0+(C_{p,v}-C_{p,l})T]$$
积分得到：
$$\boxed{\ln p=\ln p_0-\frac{L_0}{RT}+\frac{(C_{p,v}-C_{p,l})}{R}\ln T}$$

## 气液相变的麦克斯韦理论
在上面的讨论中，我们意识到一级相变必然伴随着体积的突变，这表明物态方程中有一段区域是物理上的不稳定区域，进入这一区域的物质会立刻跳到稳定区域，从而展现出了突变。对于气液相变，力学上稳定必然要求：
$$\left(\frac{\partial p}{\partial V}\right)_T<0$$
否则平衡就是力学不稳定的，这一区域便带来了一阶相变。然而除此之外，相变还要求两相压强相等与化学势相等。沿着同一条等温线，有：
$$\mathrm{d} \mu=v\mathrm{d} p$$
因此纯液相点P与纯气相点Q应该满足：
$$\mu_Q-\mu_P=\int_{v_P}^{v_Q}v\mathrm{d} p=0$$
再加上$p_P=p_Q$的条件，这给出了著名的麦克斯韦等面积法则。

## 正常-超导相变的热力学理论
许多金属和合金在温度降低到某一温度$T_c$后，发生由正常态到超导态的转变。进入超导态的金属其内的磁通量完全被排斥到其外，也即有：
$$\mathbf{B}=0$$
$$\mathbf{M}=-\mathbf{H}$$
实验还发现，当磁场达到一定值$H_c$时，超导态将被破坏。$H_c$被称为临界磁场，它是温度的函数，实验结果表明：
$$H_c(T)=H_c(0)\left[1-\left(\frac{T}{T_c}\right)^2\right]$$
因此，在$H-T$平面上，会形成一条类似于$p-V$平面的相变线。

考虑均匀各向同性的大块超导体，忽略其体积变化，则在均匀外磁场$H$中，其热力学基本微分方程为（小写字母表示单位体积）：
$$\mathrm{d} u=T\mathrm{d} s+\mu_0H\mathrm{d} M$$
作勒让德变换为：
$$\mathrm{d} g=-s\mathrm{d} T-\mu_0 M\mathrm{d} H$$
现在我们分别考虑超导态(s)和正常态(n)的自由能。对于超导态：
$$g_s(T, H) = g_s(T, 0) - \int_0^H \mu_0 M dH$$
代入$M = -H$:
$$g_s(T, H) = g_s(T, 0) + \int_0^H \mu_0 H dH = g_s(T, 0) + \frac{1}{2}\mu_0 H^2$$
对于正常态，由于几乎没有磁化，$M_n\approx 0$，因此其自由能几乎不随磁场变化：
$$g_n(T, H) \approx g_n(T, 0) = g_n(T)$$
在相界限$H = H_c(T)$上，两相的吉布斯自由能必须相等：
$$g_n(T) = g_s(T, H_c) = g_s(T, 0) + \frac{1}{2}\mu_0 H_c^2$$
现在我们考虑相变点的熵变，由$s = -\left(\frac{\partial g}{\partial T}\right)_H$，我们得到：
$$s_n(T) - s_s(T) = -\frac{d}{dT}[g_n(T) - g_s(T, 0)] = -\frac{1}{2}\mu_0 \frac{d(H_c^2)}{dT}$$
$$s_n - s_s = -\mu_0 H_c \frac{dH_c}{dT}$$
由于$\frac{dH_c}{dT} < 0$，故有$s_n > s_s$，这说明正常态比超导态更无序。相变潜热：
$$L = T(s_n - s_s) = -\mu_0 T H_c \frac{dH_c}{dT}$$
- 在有外加磁场时，相变发生在$T<T_c,H_c>0$处，因此有潜热，为**一级相变**。
- 在无外加磁场时，相变发生在$T=T_c,H_c=0$处，因此无潜热，为**二级相变**。
  
  进一步地，我们考虑定磁场热容$c = T\left(\frac{\partial s}{\partial T}\right)_H$的跃变：
  $$c_n - c_s = T \frac{d(s_n - s_s)}{dT} = -\mu_0 T \left[ \left(\frac{dH_c}{dT}\right)^2 + H_c \frac{d^2H_c}{dT^2} \right]$$
  在无外磁场的相变点，代入$H_c=0$，我们得到著名的**罗格斯公式**：
  $$\Delta c = c_s - c_n = \mu_0 T_c \left( \left. \frac{dH_c}{dT} \right|_{T=T_c} \right)^2$$
## 二级相变的朗道理论
