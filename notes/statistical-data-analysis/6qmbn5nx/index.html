与极大似然的关系

考虑NNN个独立的高斯随机变量yi,i=1,…,Ny_{i},i=1,\dots,Nyi​,i=1,…,N，每个yiy_{i}yi​都对应于另一个假定已知并且无误差的变量xix_{i}xi​。例如，我们可能在不同位置xix_{i}xi​对温度进行了NNN次测量T(xi)T(x_{i})T(xi​)。假设不同yiy_{i}yi​的均值λi\lambda_{i}λi​且未知，其方差σi2\sigma_{i}^{2}σi2​也不同但大小已知。NNN次测量yiy_{i}yi​可以等价地看作对NNN维随机矢量的单次测量，其联合概率密度函数为：

g(y,λ,σ2)=∏i=1N12πσi2exp⁡(−(yi−λi)22σi2)g(\mathbf{y},\mathbf{\lambda},\mathbf{\sigma^{2}})=\prod_{i=1}^{N} \frac{1}{\sqrt{ 2\pi\sigma_{i}^{2} }}\exp\left( \frac{-(y_{i}-\lambda_{i})^{2}}{2\sigma_{i}^{2}} \right) g(y,λ,σ2)=i=1∏N​2πσi2​​1​exp(2σi2​−(yi​−λi​)2​)

进一步假设真值λ\lambdaλ为xxx的函数，即λ=λ(x;θ)\lambda=\lambda(x;\mathbf{\theta})λ=λ(x;θ)，该函数依赖于未知参数θ=(θ1,…,θm)\mathbf{\theta}=(\theta_{1},\dots,\theta_{m})θ=(θ1​,…,θm​)。最小二乘法的目标便是对参数θ\mathbf{\theta}θ进行估计。

对联合概率密度函数取对数并忽略参数无关的相加项，可以得到对数似然函数：

log⁡L(θ)=−12∑i=1N(yi−λ(xi;θ))2σi2\log L(\mathbf{\theta})=-\frac{1}{2}\sum_{i=1}^{N} \frac{(y_{i}-\lambda(x_{i};\mathbf{\theta}))^{2}}{\sigma_{i}^{2}} logL(θ)=−21​i=1∑N​σi2​(yi​−λ(xi​;θ))2​

因此，这个对数似然函数的最大化等价于求使以下这个量最小化的参数θ\mathbf{\theta}θ的值：

χ2(θ)=∑i=1N(yi−λ(xi;θ))2σi2\chi^{2}(\mathbf{\theta})=\sum_{i=1}^{N} \frac{(y_{i}-\lambda(x_{i};\mathbf{\theta}))^{2}}{\sigma_{i}^{2}} χ2(θ)=i=1∑N​σi2​(yi​−λ(xi​;θ))2​

这就是最小二乘法的基础。即便每个yiy_{i}yi​不服从高斯分布，但只要它们之间互相独立，也可以这么定义。

如果测量不相互独立，但测量值可以用NNN维高斯概率密度函数描述，并且已知该分布的协方差矩阵VVV，而分布的均值未知，则相应的对数似然函数由多维高斯分布的联合概率密度函数的对数给出：

log⁡L(θ)=−12∑i,j=1N(yi−λ(xi;θ))(V−1)ij(yj−λ(xj;θ))\log L(\mathbf{\theta})=-\frac{1}{2}\sum_{i,j=1}^{N} (y_{i}-\lambda(x_{i};\mathbf{\theta}))(V^{-1})_{ij}(y_{j}-\lambda(x_{j};\mathbf{\theta})) logL(θ)=−21​i,j=1∑N​(yi​−λ(xi​;θ))(V−1)ij​(yj​−λ(xj​;θ))

该对数似然函数的最大化等价于求下面这个量的最小值：

χ2(θ)=∑i,j=1N(yi−λ(xi;θ))(V−1)ij(yj−λ(xj;θ))\chi^{2}(\mathbf{\theta})=\sum_{i,j=1}^{N} (y_{i}-\lambda(x_{i};\mathbf{\theta}))(V^{-1})_{ij}(y_{j}-\lambda(x_{j};\mathbf{\theta})) χ2(θ)=i,j=1∑N​(yi​−λ(xi​;θ))(V−1)ij​(yj​−λ(xj​;θ))

使得χ2\chi^{2}χ2取最小值的参数θ^1,…,θ^m\hat{\theta}_{1},\dots,\hat{\theta}_{m}θ^1​,…,θ^m​称为最小二乘估计量。在后面我们会讨论，某些条件下得到的最小χ2\chi^{2}χ2服从你卡方分布。出于这种原因，前面定义的量通常称为卡方(χ2\chi^{2}χ2)。

线性最小二乘拟合

尽管对任意函数λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)都可以应用最小二乘法，但是当λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)为参数θ\mathbf{\theta}θ的线性函数时，得到的χ2\chi^{2}χ2和最小二乘估计量具有特别理想的性质。设：

λ(x;θ)=∑j=1maj(x)θj\lambda(x;\mathbf{\theta})=\sum_{j=1}^{m} a_{j}(x)\theta_{j} λ(x;θ)=j=1∑m​aj​(x)θj​

其中aj(x)a_{j}(x)aj​(x)是xxx的任意线性独立函数。在这种情况下，我们可以解析求解最小二乘估计量及其方差， 且线性最小二乘估计量的偏倚为零，并且方差最小。

函数λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)在xix_{i}xi​处的值记作：

λ(xi;θ)=∑j=1maj(xi)θj=∑j=1mAijθj\lambda(x_{i};\mathbf{\theta})=\sum_{j=1}^{m} a_{j}(x_{i})\theta_{j}=\sum_{j=1}^{m} A_{ij}\theta_{j} λ(xi​;θ)=j=1∑m​aj​(xi​)θj​=j=1∑m​Aij​θj​

因此，χ2\chi^{2}χ2的一般表达式可以写为矩阵形式：

χ2=(y−Aθ)TV−1(y−Aθ)\chi^{2}=(\mathbf{y}-A\mathbf{\theta})^{T}V^{-1}(\mathbf{y}-A\mathbf{\theta}) χ2=(y−Aθ)TV−1(y−Aθ)

为了求解χ2\chi^{2}χ2的最小值，令其对参数θi\theta_{i}θi​的导数为0：

∇χ2=−2(ATV−1y−ATV−1Aθ)=0\nabla \chi^{2}=-2(A^{T}V^{-1}\mathbf{y}-A^{T}V^{-1}A\mathbf{\theta})=0 ∇χ2=−2(ATV−1y−ATV−1Aθ)=0

则可以求解出估计量θ^\hat{\mathbf{\theta}}θ^：

θ^=(ATV−1A)−1ATV−1y≡By\mathbf{\hat{\theta}}=(A^{T}V^{-1}A)^{-1}A^{T}V^{-1}\mathbf{y}\equiv B\mathbf{y} θ^=(ATV−1A)−1ATV−1y≡By

也就是说，求解得到的估计量θ^\hat{\mathbf{\theta}}θ^是原始测量量y\mathbf{y}y的线性函数。由误差传递的相关知识，可以得到估计量的协方差矩阵：

U=BVBT=(ATV−1A)−1U=BVB^{T}=(A^{T}V^{-1}A)^{-1} U=BVBT=(ATV−1A)−1

等价地，协方差矩阵的逆U−1U^{-1}U−1可以表示为：

(U−1)ij=12[∂2χ2∂θi∂θj]θ=θ^(U^{-1})_{ij}=\frac{1}{2}\left[ \frac{ \partial^{2}\chi^{2} }{ \partial \theta_{i}\partial\theta_{j} } \right]_{\mathbf{\theta}=\hat{\mathbf{\theta}}} (U−1)ij​=21​[∂θi​∂θj​∂2χ2​]θ=θ^​

特别地，当yiy_{i}yi​服从高斯分布时，log⁡L=−χ22\log L=-\frac{\chi^{2}}{2}logL=−2χ2​，此时上式所给出的结果与RCF边界一致。

对于λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)是参数θ\mathbf{\theta}θ的线性函数的情形，χ2\chi^{2}χ2是θ\mathbf{\theta}θ的二次型函数：

χ2(θ)=χ2(θ^)+12∑i,j=1m[∂2χ2∂θi∂θj]θ=θ^(θi−θ^i)(θj−θ^j)\chi^{2}(\mathbf{\theta})=\chi^{2}(\hat{\mathbf{\theta}})+\frac{1}{2}\sum_{i,j=1}^{m} \left[ \frac{ \partial^{2}\chi^{2} }{ \partial \theta_{i}\partial\theta_{j} } \right]_{\mathbf{\theta}=\hat{\mathbf{\theta}}}(\theta_{i}-\hat{\theta}_{i})(\theta_{j}-\hat{\theta}_{j}) χ2(θ)=χ2(θ^)+21​i,j=1∑m​[∂θi​∂θj​∂2χ2​]θ=θ^​(θi​−θ^i​)(θj​−θ^j​)

χ2(θ)=χ2(θ^)+1=χmin⁡2+1\chi^{2}(\mathbf{\theta})=\chi^{2}(\hat{\mathbf{\theta}})+1=\chi_{\min}^{2}+1 χ2(θ)=χ2(θ^)+1=χmin2​+1
