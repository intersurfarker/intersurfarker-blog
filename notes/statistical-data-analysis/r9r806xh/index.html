概率与随机变量

概率的解释

本节我们明确如何解释样本空间中的元素，以及如何为之赋予概率值并加以解释。

概率的相对频率解释

在数据分析中，概率通常解释为相对频率极限。在这种解释下，集合SSS的元素对应于测量的各种可能结果，并且认为测量可重复。SSS的子集AAA对应于子集的任何一个结果的发生。这样的子集称为一个事件。如果测量结果在该子集中，则称该事件发生。

如果SSS的子集只包含一个元素，表示单一的元输出。元输出AAA的概率指定为：当重复实验次数nnn趋于无穷时，AAA发生的次数的比率，即：

P(A)=lim⁡n→∞n次测量中A发生的次数nP(A)=\lim_{ n \to \infty } \frac{n\text{次测量中}A\text{发生的次数}}{n} P(A)=n→∞lim​nn次测量中A发生的次数​

如果AAA不是元子集，包含多个元输出，那么，这几个输出中任意一个发生的总概率就是AAA发生的概率。

主观概率

主观概率也被称作贝叶斯概率。在这种解释下，样本空间的元素对应于假设或者命题。人们把与假设关联的概率解释为可信性的量度，即：

P(A)=假设A为真的信心程度P(A)=\text{假设}A\text{为真的信心程度} P(A)=假设A为真的信心程度

在主观概率解释中，构造的样本空间中的元假设必须互斥。如果某个子集包含了多个假设，则当子集中任意一个假设为真时该子集就为真。
