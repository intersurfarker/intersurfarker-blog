与极大似然的关系

对联合概率密度函数取对数并忽略参数无关的相加项，可以得到对数似然函数：

log⁡L(θ)=−12∑i=1N(yi−λ(xi;θ))2σi2\log L(\mathbf{\theta})=-\frac{1}{2}\sum_{i=1}^{N} \frac{(y_{i}-\lambda(x_{i};\mathbf{\theta}))^{2}}{\sigma_{i}^{2}} logL(θ)=−21​i=1∑N​σi2​(yi​−λ(xi​;θ))2​

这就是最小二乘法的基础。即便每个yiy_{i}yi​不服从高斯分布，但只要它们之间互相独立，也可以这么定义。

如果测量不相互独立，但测量值可以用NNN维高斯概率密度函数描述，并且已知该分布的协方差矩阵VVV，而分布的均值未知，则相应的对数似然函数由多维高斯分布的联合概率密度函数的对数给出：

该对数似然函数的最大化等价于求下面这个量的最小值：

χ2(θ)=∑i,j=1N(yi−λ(xi;θ))(V−1)ij(yj−λ(xj;θ))\chi^{2}(\mathbf{\theta})=\sum_{i,j=1}^{N} (y_{i}-\lambda(x_{i};\mathbf{\theta}))(V^{-1})_{ij}(y_{j}-\lambda(x_{j};\mathbf{\theta})) χ2(θ)=i,j=1∑N​(yi​−λ(xi​;θ))(V−1)ij​(yj​−λ(xj​;θ))

使得χ2\chi^{2}χ2取最小值的参数θ^1,…,θ^m\hat{\theta}_{1},\dots,\hat{\theta}_{m}θ^1​,…,θ^m​称为最小二乘估计量。在后面我们会讨论，某些条件下得到的最小χ2\chi^{2}χ2服从你卡方分布。出于这种原因，前面定义的量通常称为卡方(χ2\chi^{2}χ2)。

线性最小二乘拟合

尽管对任意函数λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)都可以应用最小二乘法，但是当λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)为参数θ\mathbf{\theta}θ的线性函数时，得到的χ2\chi^{2}χ2和最小二乘估计量具有特别理想的性质。设：

λ(x;θ)=∑j=1maj(x)θj\lambda(x;\mathbf{\theta})=\sum_{j=1}^{m} a_{j}(x)\theta_{j} λ(x;θ)=j=1∑m​aj​(x)θj​

其中aj(x)a_{j}(x)aj​(x)是xxx的任意线性独立函数。在这种情况下，我们可以解析求解最小二乘估计量及其方差， 且线性最小二乘估计量的偏倚为零，并且方差最小。

函数λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)在xix_{i}xi​处的值记作：

λ(xi;θ)=∑j=1maj(xi)θj=∑j=1mAijθj\lambda(x_{i};\mathbf{\theta})=\sum_{j=1}^{m} a_{j}(x_{i})\theta_{j}=\sum_{j=1}^{m} A_{ij}\theta_{j} λ(xi​;θ)=j=1∑m​aj​(xi​)θj​=j=1∑m​Aij​θj​

因此，χ2\chi^{2}χ2的一般表达式可以写为矩阵形式：

χ2=(y−Aθ)TV−1(y−Aθ)\chi^{2}=(\mathbf{y}-A\mathbf{\theta})^{T}V^{-1}(\mathbf{y}-A\mathbf{\theta}) χ2=(y−Aθ)TV−1(y−Aθ)

∇χ2=−2(ATV−1y−ATV−1Aθ)=0\nabla \chi^{2}=-2(A^{T}V^{-1}\mathbf{y}-A^{T}V^{-1}A\mathbf{\theta})=0 ∇χ2=−2(ATV−1y−ATV−1Aθ)=0

则可以求解出估计量θ^\hat{\mathbf{\theta}}θ^：

θ^=(ATV−1A)−1ATV−1y≡By\mathbf{\hat{\theta}}=(A^{T}V^{-1}A)^{-1}A^{T}V^{-1}\mathbf{y}\equiv B\mathbf{y} θ^=(ATV−1A)−1ATV−1y≡By

也就是说，求解得到的估计量θ^\hat{\mathbf{\theta}}θ^是原始测量量y\mathbf{y}y的线性函数。由误差传递的相关知识，可以得到估计量的协方差矩阵：

U=BVBT=(ATV−1A)−1U=BVB^{T}=(A^{T}V^{-1}A)^{-1} U=BVBT=(ATV−1A)−1

(U−1)ij=12[∂2χ2∂θi∂θj]θ=θ^(U^{-1})_{ij}=\frac{1}{2}\left[ \frac{ \partial^{2}\chi^{2} }{ \partial \theta_{i}\partial\theta_{j} } \right]_{\mathbf{\theta}=\hat{\mathbf{\theta}}} (U−1)ij​=21​[∂θi​∂θj​∂2χ2​]θ=θ^​

特别地，当yiy_{i}yi​服从高斯分布时，log⁡L=−χ22\log L=-\frac{\chi^{2}}{2}logL=−2χ2​，此时上式所给出的结果与RCF边界一致。

对于λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)是参数θ\mathbf{\theta}θ的线性函数的情形，χ2\chi^{2}χ2是θ\mathbf{\theta}θ的二次型函数：

χ2(θ)=χ2(θ^)+1=χmin⁡2+1\chi^{2}(\mathbf{\theta})=\chi^{2}(\hat{\mathbf{\theta}})+1=\chi_{\min}^{2}+1 χ2(θ)=χ2(θ^)+1=χmin2​+1

分区间数据的最小二乘拟合

事实上，联系真值λ\lambdaλ和变量xxx的函数不一定是xxx的概率密度函数，而可以是任意函数。例如，假设我们对随机变量xxx进行nnn次测量，并做成分NNN个区间的直方图。设yiy_{i}yi​为区间iii的频数，f(x;θ)f(x;\mathbf{\theta})f(x;θ)为假设的概率密度函数，我们需要估计θ=(θ1,…,θm)\mathbf{\theta}=(\theta_{1},\dots,\theta_{m})θ=(θ1​,…,θm​)。于是，区间iii的预期事例数λi=E[yi]\lambda_{i}=E[y_{i}]λi​=E[yi​]为：

λi(θ)=n∫ximin⁡ximax⁡f(x;θ) dx=npi(θ)\lambda_{i}(\mathbf{\theta})=n\int_{x_{i}^{\min}}^{x_{i}^{\max}} f(x;\mathbf{\theta}) \, dx=np_{i}(\mathbf{\theta}) λi​(θ)=n∫ximin​ximax​​f(x;θ)dx=npi​(θ)

参数θ\mathbf{\theta}θ可以通过最小化下面这个量求得：

χ2(θ)=∑i=1N(yi−λi(θ))2σi2\chi^{2}(\mathbf{\theta})=\sum_{i=1}^{N} \frac{(y_{i}-\lambda_{i}(\mathbf{\theta}))^{2}}{\sigma_{i}^{2}} χ2(θ)=i=1∑N​σi2​(yi​−λi​(θ))2​

如果每个区间的平均事例数相比总事例数很小，则每个区间的事例数近似服从泊松分布，此时方差即为均值：

χ2(θ)=∑i=1N(yi−λi(θ))2λi(θ)=∑i=1N(yi−npi(θ))2npi(θ)\chi^{2}(\mathbf{\theta})=\sum_{i=1}^{N} \frac{(y_{i}-\lambda_{i}(\mathbf{\theta}))^{2}}{\lambda_{i}(\mathbf{\theta})}=\sum_{i=1}^{N} \frac{(y_{i}-np_{i}(\mathbf{\theta}))^{2}}{np_{i}(\mathbf{\theta})} χ2(θ)=i=1∑N​λi​(θ)(yi​−λi​(θ))2​=i=1∑N​npi​(θ)(yi​−npi​(θ))2​

另一种方法是，用区间iii的实际观测事例数近似相应的方差，而不用预期值λi(θ)\lambda_{i}(\mathbf{\theta})λi​(θ)。这就是所谓的修正的最小二乘法。修正的最小二乘法要求对下式求最小值：

χ2(θ)=∑i=1N(yi−λi(θ))2yi=∑i=1N(yi−npi(θ))2yi\chi^{2}(\mathbf{\theta})=\sum_{i=1}^{N} \frac{(y_{i}-\lambda_{i}(\mathbf{\theta}))^{2}}{y_{i}}=\sum_{i=1}^{N} \frac{(y_{i}-np_{i}(\mathbf{\theta}))^{2}}{y_{i}} χ2(θ)=i=1∑N​yi​(yi​−λi​(θ))2​=i=1∑N​yi​(yi​−npi​(θ))2​

这个方法比较容易计算，但缺点是如果某些区间事例数很少甚至没有事例，则误差估计可能很差。

在使用最小二乘法拟合直方图时，需要注意以下的问题。通常我们不使用实际观测的总事例数nnn求λi\lambda_{i}λi​，而是引入额外的可调参数ν\nuν作为归一化因子。此时区间iii的预期事例数λi(θ,ν)=E[yi]\lambda_{i}(\mathbf{\theta},\nu)=E[y_{i}]λi​(θ,ν)=E[yi​]为：

λi(θ,ν)=νpi(θ)\lambda_{i}(\mathbf{\theta},\nu)=\nu p_{i}(\mathbf{\theta}) λi​(θ,ν)=νpi​(θ)

然而如果我们引入可调归一化参数，考虑用上面两种χ2\chi^{2}χ2对ν\nuν求极值，可以得到估计量：

ν^LS=n+χ22\hat{\nu}_{\text{LS}}=n+\frac{\chi^{2}}{2} ν^LS​=n+2χ2​

ν^MLS=n−χ2\hat{\nu}_{\text{MLS}}=n-\chi^{2} ν^MLS​=n−χ2

一般而言，每个区间对χ2\chi^{2}χ2的贡献约为1，因此对于最小二乘法，总事例数会产生N/nN /nN/n量级的误差。根本上，这是由于我们高斯分布的假设是不准确的。我们可以将分区间数据的最小二乘法与极大似然法相比较。在极大似然法中，各区间事例数yiy_{i}yi​的联合概率密度函数取为多项分布，或者也可以认为yiy_{i}yi​是泊松随机变量。在后一种情形中，期待的总事例数ν\nuν被当作可调参数，且可以自动得到正确的估计量ν^=n\hat{\nu}=nν^=n。此外，极大似然估计量的方差也比最小二乘法更快地收敛于最小方差边界。因此，在直方图拟合中最好采用最大似然法。

拟合优度最小二乘检验

前面已经提到，如果测量值yiy_{i}yi​服从高斯分布，则最小二乘估计量与极大似然估计量一致，并且χ2\chi^{2}χ2值可以用来检验假设为真的条件下得到观测数据的可能性有多大。可以证明，如果：

  1. yi(i=1,…,N)y_{i}(i=1,\dots,N)yi​(i=1,…,N)为方差σi2\sigma_{i}^{2}σi2​已知的相互独立的高斯随机变量；
  2. λ(x;θ1,…,θm)\lambda(x;\theta_{1},\dots,\theta_{m})λ(x;θ1​,…,θm​)这个假设是参数θi\theta_{i}θi​的线性函数；
  3. 假设的函数形式是正确的； 那么上面定义的最小χ2\chi^{2}χ2值服从自由度为N−mN-mN−m的卡方分布。

服从卡方分布的随机变量zzz的期望值等于自由度的数目，因此经常用χ2/nd\chi^{2} /n_{d}χ2/nd​来衡量拟合优度。如果这个值在1附近，则一切都符合预期；如果比值远小于1，则需要检查误差σi\sigma_{i}σi​是否被高估或存在相关性。而如果远大于1，则有理由怀疑假设的正确性。

通常为给定的χ2\chi^{2}χ2提供一个显著水平(P-值)，用于表示当前假设导致更差χ2\chi^{2}χ2的概率，也即：

P=∫χ2∞f(z;nd) dzP=\int_{\chi^{2}}^{\infty} f(z;n_{d}) \, dz P=∫χ2∞​f(z;nd​)dz

P-值多大时可以决定拒绝某个假设有一定的主观性，但需要注意的是，低估误差σ2\sigma^{2}σ2会使正确的假设给出很差的χ2\chi^{2}χ2值。

应当注意小统计误差和好的χ2\chi^{2}χ2值之间的区别：统计误差反映了参数离开拟合值时χ2\chi^{2}χ2的变化，而与χ2\chi^{2}χ2本身的绝对值无关。统计误差仅仅依赖于复合假设λ(x;θ)\lambda(x;\mathbf{\theta})λ(x;θ)和原始测量的协方差矩阵VVV，而与测量值yiy_{i}yi​无关。

用最小二乘法合并实验测量

最小二乘法的一个特殊应用是合并对同一个量的很多不同测量。假设对某个真值λ\lambdaλ未知的量进行了NNN次测量，得到了相互独立的测量值yiy_{i}yi​和估计误差σi\sigma_{i}σi​。由于我们假定对所有测量而言真值为常数，故卡方为：

χ2(λ)=∑i=1N(yi−λ)2σi2\chi^{2}(\lambda)=\sum_{i=1}^{N} \frac{(y_i-\lambda)^{2}}{\sigma_{i}^{2}} χ2(λ)=i=1∑N​σi2​(yi​−λ)2​

λ^=∑i=1Nyi/σi2∑j=1N1/σj2\hat{\lambda}=\frac{\sum_{i=1}^{N} y_{i} /\sigma_{i}^{2}}{\sum_{j=1}^{N} 1 /\sigma_{j}^{2}} λ^=∑j=1N​1/σj2​∑i=1N​yi​/σi2​​

存在相关性的测量数据的平均

考虑用两个不同的尺子测量物体的长度，它们用不同的材料制作，因此热膨胀不同。假设两个尺子在T0T_{0}T0​温度下都是准确的，但在其它温度，真实长度λ\lambdaλ的修正估计需要通过下式得到：

yi=Li+ci(T−T0)y_{i}=L_{i}+c_{i}(T-T_{0}) yi​=Li​+ci​(T−T0​)

将温度TTT的测量值视为标准差为σT\sigma_{T}σT​的高斯随机变量，并假定两次测量的TTT相同。将未修正的测量值LiL_{i}Li​视作标准差为σLi\sigma_{L_{i}}σLi​​的随机变量，并假定ci,σLi,σTc_{i},\sigma_{L_{i}},\sigma_{T}ci​,σLi​​,σT​都已知。

ρ=V12σ1σ2=c1c2σT2(σL12+c12σT2)(σL22+c22σT2)\rho=\frac{V_{12}}{\sigma_{1}\sigma_{2}}=\frac{c_{1}c_{2}\sigma_{T}^{2}}{\sqrt{ (\sigma_{L_{1}}^{2}+c_{1}^{2}\sigma_{T}^{2})(\sigma_{L_{2}}^{2}+c_{2}^{2}\sigma_{T}^{2}) }} ρ=σ1​σ2​V12​​=(σL1​2​+c12​σT2​)(σL2​2​+c22​σT2​)​c1​c2​σT2​​

而其方差为0。本质上，出现负权重的原因是，在已经非常精确地知道LiL_{i}Li​的值的情况下，如果y1y_{1}y1​与y2y_{2}y2​仍然相差很大，那唯一的可能就是真实温度与测量值TTT相差较大。因此，加权平均λ^\hat{\lambda}λ^被拖向两条阴影带y1(T)y_{1}(T)y1​(T)和y2(T)y_{2}(T)y2​(T)的交叉点。

需要问的一个问题是，在实际应用中我们在多大程度上可以信赖这两个前提。两次测量实际上部分基于共同的测量，即对温度的测量(温度本身也是一个随机变量)，这是相关性的根源所在。在对高度相关的量作平均时，我们需要倍加小心，因为每次测量之间的协方差矩阵中很小的不确定度都会导致平均值λ^\hat{\lambda}λ^出现很大的不确定度，并且其方差估计也不再正确。当不确定度的相关性被高估时，尤其如此。

求解协方差矩阵

为了用最小二乘法对结果进行平均，需要知道协方差矩阵VVV。对于简单的情形，我们可以直接计算协方差矩阵，例如：如果两次测量仅仅部分而非全部使用相同数据。比如考虑对某个量xxx的独立观测，xxx本身的方差为σ2\sigma^{2}σ2。假设第一次测量y1y_{1}y1​由nnn各独立事例的样本均值构造而成：

y1=1n∑i=1nxiy_{1}=\frac{1}{n}\sum_{i=1}^{n} x_{i} y1​=n1​i=1∑n​xi​

第二次测量y2y_{2}y2​用了mmm个xxx值：

y2=1m∑i=1mxiy_{2}=\frac{1}{m}\sum_{i=1}^{m} x_{i} y2​=m1​i=1∑m​xi​

其中有ccc个共用值。可以求得：

cov[y1,y2]=cσ2nm\mathrm{cov}[y_{1},y_{2}]=\frac{c\sigma^{2}}{nm} cov[y1​,y2​]=nmcσ2​

或者等价地，ρ=cnm\rho=\frac{c}{\sqrt{ nm }}ρ=nm​c​。在这个例子中：

ρ=cmn=σ1σ2cm≤σ1σ2\rho=\frac{c}{\sqrt{ mn }}=\frac{\sigma_{1}}{\sigma_{2}} \frac{c}{m}\leq \frac{\sigma_{1}}{\sigma_{2}} ρ=mn​c​=σ2​σ1​​mc​≤σ2​σ1​​

因此不会出现负权重。

在更一般的情况下，确定协方差矩阵可能需要蒙特卡罗方法进行大量模拟。但即便不知道协方差矩阵，我们仍然可以任意选取权重ωi\omega_{i}ωi​，只要保证其和为1，得到的估计量λ^\hat{\lambda}λ^就是无偏的。但这种情况下，将无法得到最小可能的方差，平均值的方差估计将不再正确。
